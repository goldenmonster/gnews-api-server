## GNews API backend

## HOW TO RUN

## Get Started

```bash
# copy .env
cp .env.example
```

```bash
# install dependencies
npm install
```

Or use `yarn`

```bash
yarn install
```

```bash
# run project for development
yarn dev
```

```bash
# run project
yarn start
```

## Features

- API for top trending news

- API for searching news by title, description and pageniation

# Technical details

- Express.js

- GNews api integration

- API caching
